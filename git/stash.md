# git stash

## Save changes

```bash
git stash save "mensaje opcional para ti"
```

## Save change with a message

```bash
git stash save "mensaje opcional para ti"
```

## View changes

```bash
git stash list
```

## Recover Changes and keep it in stash

```bash
git stash apply
```

## Recover Changes and remove fromstash

```bash
git stash pop
```
